# Sprint 9.11 — UI-14 Badge: badge mềm cho tất cả status

Em tiếp tục Sprint 9. **Sprint con 9.11 = UI-14**.

## BA spec exact (sheet "5. Issue List" cột G + H — xlsm 2026-05-22)

- **Khu vực:** Badge
- **Vấn đề (cột E):** Status hiện tại có chỗ là text thường như "Draft", chưa dùng badge.
- **Đề xuất điều chỉnh (cột G):** Tất cả status phải dùng **badge mềm**: Draft, Đang xử lý, Cần phản hồi, Hoàn tất…
- **Kết quả mong muốn (cột H):** Không còn status dạng text thường trong table/list.

**=> Audit toàn bộ template → mỗi chỗ render status là raw text → wrap bằng `.wujia-badge` + variant phù hợp.**

## File chạm (dự kiến)

- Mỗi template trong các module portal có render status → wrap:
  ```xml
  <span t-attf-class="wujia-badge wujia-badge-#{record.state == 'done' and 'success' or record.state == 'draft' and 'muted' or 'info'}"
        t-esc="record.state_label or record.state"/>
  ```
  hoặc đơn giản hơn: dùng class `state-badge state-#{record.state}` (auto-upgrade đã có trong `_components.css`).
- Map status → variant (English-only, theo BA Issue List ví dụ):
  - Draft → `muted`
  - Submitted / Đang xử lý → `info`
  - Cần phản hồi / Awaiting Reply → `warning`
  - Hoàn tất / Done → `success`
  - Reject / Cancelled → `danger`
