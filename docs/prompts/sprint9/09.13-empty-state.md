# Sprint 9.13 — Empty state: icon + text + spacing chuẩn

Em tiếp tục Sprint 9. **Sprint con 9.13 = Empty state** (BA không đặt ID, gọi tên "Empty state").

## BA spec exact (sheet "5. Issue List" cột G + H — xlsm 2026-05-22)

- **Khu vực:** Empty state
- **Vấn đề (cột E):** Empty state "Chưa có dữ liệu" còn đơn giản, hơi thô.
- **Đề xuất điều chỉnh (cột G):** Dùng **empty state thống nhất**: **icon nhỏ + dòng text + khoảng trắng chuẩn**. Ví dụ: "Chưa có đơn hàng nào."
- **Kết quả mong muốn (cột H):** Empty state nhìn gọn, không làm card bị trống khó chịu.

**=> 3 element:** icon + text + spacing (padding around).
