# Sprint 9.5 — UI-05 Button: chuẩn hoá Primary + Secondary

Em tiếp tục Sprint 9. **Sprint con 9.5 = UI-05**.

## BA spec exact (sheet "5. Issue List" cột G + H — xlsm 2026-05-22)

- **Khu vực:** Button
- **Vấn đề (cột E):** Chuẩn hoá lại giao diện các button trên portal.
- **Đề xuất điều chỉnh (cột G):** (BA không ghi)
- **Kết quả mong muốn (cột H):** Button **primary: nền xanh, chữ trắng, height 40–44px**. Button **secondary: nền trắng, viền xám, height 36–40px**.
- **Ghi chú (cột I):** Button cùng loại phải giống nhau trên mọi page.

**=> CHỈ 2 thuộc tính secondary: nền trắng + viền xám + height 36-40. BA KHÔNG nói "text xám" — KHÔNG bịa.** Color text secondary giữ default Bootstrap (text-body / text-primary tuỳ component cha) hoặc dùng `--wujia-text-primary` cho readable.

## Bước verify trước khi code

1. Mở xlsm row UI-05 → confirm cột H + I verbatim.
2. Grep button hiện trạng:
   ```bash
   grep -n "btn-primary\|btn-secondary\|wujia-btn\|btn-outline" custom/wujia_portal_layout/static/assets/css/_components.css
   grep -rn "btn btn-primary\|btn btn-secondary\|btn-outline-primary\|wujia-btn-secondary" custom/*/views/ custom/*/templates/
   ```
3. Xem `--wujia-btn-height` hiện tại trong `_variables.css` (42px — phù hợp primary range 40-44, OK).
4. Audit: tất cả page dùng button đều dùng class `.btn .btn-primary` hoặc `.btn .wujia-btn-secondary` thống nhất chưa? Nếu có chỗ inline style hex → ghi nhận để fix.

## File chạm

- `custom/wujia_portal_layout/static/assets/css/_variables.css` — bổ sung `--wujia-btn-height-secondary: 38px` (mid range 36-40).
- `custom/wujia_portal_layout/static/assets/css/_components.css`:
  - `.btn` keep `min-height: var(--wujia-btn-height)` (42px) — primary range.
  - `.btn-outline-primary, .wujia-btn-secondary` thêm `min-height: var(--wujia-btn-height-secondary)` (38px) + `background-color: #FFFFFF !important` + `border: 1px solid var(--wujia-border) !important` (xám = `#E5E7EB`).
  - **KHÔNG** override `color` (giữ default readable).
- (nếu audit phát hiện) Các template còn dùng `style="background:..."` inline → đổi sang class.
