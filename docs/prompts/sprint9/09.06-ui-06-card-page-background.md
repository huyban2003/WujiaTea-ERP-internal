2. Xem giá trị hiện tại:
   ```bash
   grep -n "bg-page\|bg-card\|F5F7FA\|F6F8FA" custom/wujia_portal_layout/static/assets/css/_variables.css
   ```
   Hiện tại: `--wujia-bg-page: #F5F7FA` (đúng) + `--wujia-bg-card: #FFFFFF` (đúng). Có thể đã DONE từ Sprint 8 token — nhưng BA vẫn list lại issue này có nghĩa **render thực tế chưa đậm**. Kiểm tra apply.

   ```bash
   grep -rn "background-color: var(--wujia-bg-page)" custom/wujia_portal_layout/static/assets/css/
   grep -rn "background-color\|bg-light\|#fff\|#FFF" custom/*/static/src/css/ custom/*/static/assets/css/
   ```
4. Inspect DevTools `/portal`: `body` có applied `--wujia-bg-page` chưa? Có module nào override bằng `bg-light` (#f8f9fa) lighter không?
