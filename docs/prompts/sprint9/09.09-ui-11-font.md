Em tiếp tục Sprint 9. **Sprint con 9.9 = UI-11**.

## BA spec exact (sheet "5. Issue List" cột G + H — xlsm 2026-05-22)

- **Khu vực:** Font
- **Vấn đề (cột E):** Font size và độ đậm chưa thống nhất, một số text hơi nhỏ/mảnh.
- **Đề xuất điều chỉnh (cột G):** Dùng **font Inter/Arial**. **Body 15–16px, title 32–36px, card title 20–22px, table text 15–16px**.

**=> 1 font-family + 4 size token.**
