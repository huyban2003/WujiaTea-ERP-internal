# Sprint 9.12 — UI-15 Responsive: desktop 2 cột, mobile drawer/modal

Em tiếp tục Sprint 9. **Sprint con 9.12 = UI-15**.

## BA spec exact (sheet "5. Issue List" cột G + H — xlsm 2026-05-22)

- **Khu vực:** Responsive
- **Vấn đề (cột E):** Chưa đảm bảo mobile/tablet. Màn Support có detail panel bên phải dễ bị cắt nếu màn nhỏ.
- **Đề xuất điều chỉnh (cột G):** **Desktop dùng 2 cột**. **Tablet/mobile: detail panel xuống dưới hoặc mở dạng drawer/modal**.
- **Kết quả mong muốn (cột H):** Không còn block bị cắt ngang khi giảm width.

**=> 2 mode:**
- Desktop ≥ 992px: 2 cột master-detail.
- < 992px: detail xuống dưới (stack) HOẶC mở dạng drawer/modal khi click item list.

## Workflow

1. Read xlsm verify spec.
2. Audit page (bước 3-4 trên) — báo cáo anh số page có 2-pane.
3. Đề xuất Option A vs B — hỏi anh chọn:
   - A: stack (đã hỗ trợ, chỉ cần wrap class). Trade-off: detail dài → user scroll xa.
   - B: drawer overlay. Trade-off: thêm JS, cần show/hide trigger.
4. Anh chọn → plan ngắn → approve.
5. Edit → `bash scripts/upgrade.sh "<csv module>"` RC=0.
6. Restart Odoo → test 3 viewport (360, 768, 1280) cho mỗi page có 2-pane → screenshot.
7. Không còn block bị cắt ngang. Master list usable, detail accessible.
8. Anh OK → §9 bảng UI-15 ⬜ → ✅.
