\begin{itemize}
  \item \textbf{Sản phẩm}: title-row (tiêu đề 24px + \texttt{wj-count-meta} ``N SP'') / ô tìm kiếm
        tách riêng full-width / \textbf{chips danh mục} thay dropdown (\texttt{wj-filter-chips} +

        category, giữ \texttt{keyword} qua querystring + hidden input) / mỗi sản phẩm là
        \textbf{1 card riêng} (gap 6, border \texttt{\#EEF2F5}, h58) / thumb hiện \textbf{text ĐVT}
        thay icon / badge ``đã trong giỏ'' đổi \textbf{green \(\rightarrow\) cyan} / nút thêm 46×32.
  \item \textbf{Giỏ hàng}: title-row + count / back-button gọn 38px / \textbf{warnbar khung giờ MỚI}
        (reuse \texttt{.wujia-morder-warnbar}, gate \texttt{order\_window\_enabled}) / mỗi dòng giỏ
        là 1 card (h76, gap 6) thumb 40 text ĐVT, meta ``ĐVT · Đơn giá X'' 1 dòng, stepper compact

        summary \textbf{giữ sticky} restyle (Tạm tính/Số lượng + submit full-width).
  \item \textbf{BE}: \texttt{portal\_order\_cart\_view} thêm \texttt{order\_window\_enabled} +
        \texttt{order\_time\_from/to} (copy pattern catalog, tái dùng \texttt{\_float\_to\_hhmm} +

        JS \texttt{portal\_order.js} \textbf{không đổi} (giữ class/data-attr: add-cart, stepper,
        del, floatbar count). Chips = link server round-trip, không cần JS.
\end{itemize}
