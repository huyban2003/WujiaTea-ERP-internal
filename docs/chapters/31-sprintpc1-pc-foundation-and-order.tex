\section{Bối cảnh}

Mobile UI đã xong (Sprint 10--21). Chuyển sang \textbf{desktop/PC}, vốn bị ``GIỮ NGUYÊN 100\%''
suốt các sprint mobile (mọi block bọc \texttt{d-none d-lg-block}, không đổi từ Sprint 9). BA giao bộ
source desktop \texttt{pc\_source\_ui\_v1\_4} (12 component SVG + screen template + token + QA
checklist). Full PC reconcile = 12 component \(\times\) \(\sim\)9 trang \(\rightarrow\) quá lớn cho 1
sprint; user chốt làm theo workflow BA: \textbf{Part A} dựng component library trước, \textbf{Part B}
ráp screen đầu tiên (Đặt hàng), các trang còn lại ``từng trang sửa sau''.

\textbf{Token strategy (chốt):} scoped \texttt{wj-pc-*} (zero blast radius) --- thêm token PC mới +
class \texttt{wj-pc-*} mới, \textbf{KHÔNG} bump token global \texttt{--wujia-*} (tránh vỡ mobile vừa
xong). Conflict BA chỉ áp \textbf{Sidebar active}; KHÔNG thêm cart/bell vào TopHeader; KHÔNG dựng lại
nav ``Công nợ'' (giữ quyết định xoá S17).

\section{(B) Màn Đặt hàng PC --- combined (Figma \texttt{PC\_Product\_Cart 4600:2})}

BA gộp \textbf{catalog (trái) + giỏ hàng ĐẦY ĐỦ (phải) trên 1 trang} \texttt{/portal/order} --- khác
hiện tại (catalog có cart-sidebar nhỏ + trang \texttt{/cart} riêng). Quyết định user: (1) gộp +
restyle cả \texttt{/portal/order/cart} desktop thành standalone (dùng chung partial); header cart icon
\(\rightarrow\) \texttt{/portal/order}; (2) panel chỉ \textbf{Ghi chú} (bỏ SĐT/Địa chỉ theo Figma,
submit fallback); (3) warning-bar khung giờ \textbf{wire}; min-qty toast \textbf{chỉ CSS, chưa wire}.

\begin{itemize}
  \item \textbf{Controller} (\texttt{portal.py}): \texttt{/portal/order} truyền thêm \texttt{cart} +
        \texttt{product\_count} + \texttt{page\_size} + \texttt{message}/\texttt{error}.
        \texttt{\_fallback\_pager} \(\rightarrow\) numbered \texttt{pages} + \textbf{giữ
        keyword/category khi sang trang} (sửa luôn bug cũ mất filter). \texttt{cart\_qty\_map} tính
        inline trong template (như mobile). KHÔNG thêm validation min/max (decision 3).
  \item \textbf{Partial dùng chung} (\texttt{pc\_cart\_panel.xml}): cột giỏ phải (warning-bar + bảng

        \texttt{/portal/order} (cột phải) VÀ \texttt{/portal/order/cart} desktop (standalone) \(\Rightarrow\) DRY.
  \item \textbf{Template}: \texttt{portal\_order\_catalog.xml} desktop \(\rightarrow\) combined
        (\texttt{wj-pc-page-header} + filter row + product table header \textbf{cyan} + cột Quy

  \item \textbf{CSS screen} (\texttt{portal\_order.css}): \texttt{wj-pc-order-*}/\texttt{wj-pc-cart-*} +

\end{itemize}
