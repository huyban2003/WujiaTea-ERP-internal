\section{UI-02 --- migration default lang}

DB thật lúc bắt đầu: chỉ \texttt{en\_US} active, \texttt{vi\_VN} tồn tại nhưng inactive; 6 user
group\_portal đều \texttt{en\_US}. Migration \texttt{wujia\_portal\_base/migrations/19.0.5.12.0/post-activate-vi-lang.py}:

\begin{lstlisting}[language=Python]
def migrate(cr, version):
    env = api.Environment(cr, SUPERUSER_ID, {})
    env["res.lang"]._activate_lang("vi_VN")
    portal_group = env.ref("base.group_portal", raise_if_not_found=False)
    if not portal_group:
        return
    # active_test=False -> gom cả template portal user (inactive) => user MOI cung vi_VN
    users = env["res.users"].with_context(active_test=False).search(
        [("group_ids", "in", portal_group.id), ("lang", "!=", "vi_VN")])
    if users:
        users.write({"lang": "vi_VN"})
\end{lstlisting}

Idempotent, scoped group\_portal $\to$ user backend không đụng. Chạy khi \texttt{-u wujia\_portal\_base}.
Field groups Odoo 19 = \texttt{group\_ids} (không phải \texttt{groups\_id}).
