\subsection{Logic check (BA row 167--173)}

Helper \texttt{Settings.\_is\_within\_order\_window()} trả \texttt{(allowed, window)}:

\begin{itemize}[leftmargin=*]
    \item Nếu \texttt{enabled=False} \(\Rightarrow\) always allowed.
    \item Tính \texttt{now} = giờ hiện tại trong timezone của \texttt{env.user.tz} (fallback UTC) thành float hours.
    \item Trường hợp \texttt{from <= to}: allowed nếu \(from \le now \le to\).
    \item Trường hợp \texttt{from > to} (khung qua nửa đêm): allowed nếu \(now \ge from\) hoặc \(now \le to\).
\end{itemize}
