# Wujia UI CHANGELOG — FINAL v1

## Final decisions

1. Header mobile giảm từ `112px` xuống `104px`.
2. Header dùng nền xanh liền khối `#28A9DF`, không có dải xanh đậm ở đáy.
3. Header mobile chỉ giữ `Language / Cart shortcut / Avatar`.
4. Notification bell bị loại khỏi header mobile vì footer đã có tab `Thông báo`.
5. Cart trên header là shortcut vào giỏ hàng hiện tại, không trùng với footer tab `Đặt hàng`.
6. Current Store Strip giữ `48px`, chỉ hiển thị store code, store name, role badge.
7. Footer giữ `83px`, active tab chuyển sang brand cyan `#28A9DF`.
8. Màu đỏ `#EF4444` chỉ dùng cho badge, cảnh báo, lỗi; không dùng làm màu active navigation.
9. Content area bắt buộc có padding ngang `16px` và padding bottom tối thiểu `96px` để không bị fixed footer che.
10. Source pack này thay thế các pack v1/v2/v3 trước đó.
